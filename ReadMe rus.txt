Данный проект посвящен русской локализации гоночного симулятора Assetto Corsa от компании Kunos Simulazioni. 

Для перевода контента игры на русский язык нужно перекодировать файл json в UTF-8 и после этого отредактировать его. 

Для установки данной локализации, скопируйте содержимое репозитория в папку с игрой с заменой файлов. Все пути сохранены. За основу взята лицензионная steam версия игры с последними обновлениями.
